###5. Cool dev trick!
    I use this `outline` css property to show the positioning of elements when i'm in dev phase. Makes me feel like a samaurai warrior. 
    ```
    *,
    *:before,
    *:after {
        box-sizing: border-box;
        outline: 1px solid rgba(200,0,0,.15);
    }
    ```
